# Changesets

Hello and welcome! This folder holds changesets for this repo.

When you make a change that affects the published package, run:

```bash
npx changeset
```

and follow the prompts (choose patch/minor/major and add a short description). The resulting `.md` file should be committed with your PR. On release, these drive version bumps and CHANGELOG entries.

See [CONTRIBUTING.md](../CONTRIBUTING.md) and the [Changesets docs](https://github.com/changesets/changesets).
